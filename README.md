# 🐶 TinDog

A fun and responsive landing page for a fictional dog dating app — **TinDog**. Built with **HTML**, **CSS**, and **Bootstrap 5**.

---

## 🌐 Live Demo

> https://sandeepkhanna13.github.io/tindog/

## 📸 Preview

![TinDog Preview](./images/iphone.png)

---

## 🚀 Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Sticky navbar with hamburger menu on mobile
- ✅ Hero section with App Store & Google Play buttons
- ✅ Features section with icons
- ✅ Testimonial section with press logos
- ✅ Pricing cards (Chihuahua, Labrador, Mastiff plans)
- ✅ Contact section
- ✅ Footer with navigation links

## 🏃 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/sandeepkhanna13/tindog.git
   ```

2. Navigate into the project folder:
   ```bash
   cd tindog
   ```

3. Open `index.html` in your browser:
   - Double click `index.html`, OR
   - Use **Live Server** extension in VS Code (recommended)

## 📱 Responsive Breakpoints

| Screen | Layout |
|---|---|
| Mobile (< 576px) | Single column, stacked buttons |
| Tablet (≥ 768px) | 2-column press logos, 3-column pricing |
| Desktop (≥ 992px) | Full layout with side-by-side hero |
